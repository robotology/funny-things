Copy\+Policy\+: Released under the terms of the GNU GPL v2.\+0.\hypertarget{group__icub__iCubWriter_intro_sec}{}\doxysubsection{\texorpdfstring{Description}{Description}}\label{group__icub__iCubWriter_intro_sec}
This is a module for implementing the IMU IDENTIFIER on the i\+Cub. It uses a set of waypoints in order to move the i\+Cub head to a specified position with a specified velocity. In the meanwhile, it reads both references at the PID level, and the gyro output from the inertial, sending them out through the IMU\+:o port for later log and use.\hypertarget{group__icub__iCubWriter_lib_sec}{}\doxysubsection{\texorpdfstring{Libraries}{Libraries}}\label{group__icub__iCubWriter_lib_sec}
